# Distributed Email Scheduler

A web-based application that allows users to schedule emails to be sent automatically at a specified date and time. The system uses a frontend interface for scheduling and a backend service to handle email delivery.

---

## Features

- Schedule emails for future delivery
- Automatic email sending
- Frontend built with React
- Backend built with Node.js and Express
- Clean and modular project structure
